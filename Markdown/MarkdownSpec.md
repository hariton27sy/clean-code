## Definitions
* Markdown code -> HTML code
* All in [] are additional symbols which don't influence ont the representation


## Headers
Can be used only in the start of a line
* `# h1 Header[ #*]`
    # h1 Header
---
* `## h2 Header[ #*]`
    ## h2 Header
---
* `### h3 Header[ #*]`
    ### h3 Header
---
* `#### h4 Header[ #*]`
    #### h4 Header
---
* `##### h5 Header[ #*]`
    ##### h5 Header
---
* `###### h6 Header[ #*]`
    ###### h6 Header
---

## Bold and Italic
Can be used everywhere
* `__Italic line_`<br>
_Italic line_
* `___Bold Line___**`<br>
**Bold Line**
* `___Bold italic Line___`<br>
***Bold italic line***
